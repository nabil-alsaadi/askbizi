Cookies

Our website uses ‘cookies’. These cookies are used to store information including visitors’ preferences, and the pages on the website that the visitor accessed or visited.
A cookie file can contain a variety of information, including:

  • The name of the website that created the cookie
  • The cookie's expiration date
  • The cookie's value, which is usually a string of text
  • The domain name of the website that created the cookie
  • The path of the cookie (the part of the website that the cookie is associated with)
  • The secure flag, which indicates whether the cookie should only be sent over secure connections
  • The SameSite attribute, which indicates whether the cookie should be sent to third-party websites

In addition to these basic properties, cookies can also be used to store other types of information, such as:

  • User preferences
  • Shopping cart contents
  • Login status
  • Advertising preferences
  • Tracking information

The information is used to optimize the users’ experience by customizing our web page content based on visitors’ browser type and/or other information.
